# HEARTBEAT.md — Mango's Periodic Checks + Learning Triggers

## Priority Checks (Every Heartbeat)

- [ ] Check for new client messages/inquiries — respond immediately (< 30 min)
- [ ] Check for leads needing follow-up (2-day, 5-day, 2-week marks)
- [ ] Review today's schedule — any appointments or installs?
- [ ] Scan for anything the boss sent that I haven't addressed

## Daily Checks (2-3x per day)

- [ ] Update `leads/PIPELINE.md` with any status changes
- [ ] Log interactions in `memory/YYYY-MM-DD.md`
- [ ] Check social media for messages or comments
- [ ] Review pending quotes — any that need sending or follow-up?

### Daily Learning Triggers

- [ ] **Did the boss correct me today?** If yes → immediately log in `LESSONS.md` under "Boss Preferences"
- [ ] **Did I make a mistake today?** If yes → log in `LESSONS.md` under "Mistakes I Won't Repeat"
- [ ] **Did something work unusually well?** If yes → log in `LESSONS.md` under "Patterns That Work"
- [ ] **Did I interact with a client?** If yes → update their entry in `LESSONS.md` → "Client Playbook"
- [ ] **End of day:** Write a quick reflection in today's daily log — 2-3 sentences on what went well and what to do better tomorrow

## Weekly Checks (Every Monday)

- [ ] **Write this week's entry in `GROWTH.md`** — use the template, be honest
- [ ] Review `LESSONS.md` — am I actually applying what I learned? Any patterns I'm ignoring?
- [ ] Generate weekly metrics summary for the boss
- [ ] Review pipeline — how many leads, quotes, closes this week?
- [ ] Compare revenue against $1M annual target pace
- [ ] Review marketing performance — what's working?
- [ ] Update `MEMORY.md` with any key insights worth preserving long-term
- [ ] Plan social media content for the week
- [ ] Set 3-5 specific goals for the coming week in `GROWTH.md`

## Monthly Checks

- [ ] Full financial reconciliation
- [ ] Lead source analysis — where are leads coming from?
- [ ] Close rate analysis — where are we losing deals?
- [ ] Update pricing and service area info if changed
- [ ] Review `GROWTH.md` trend — am I actually improving month over month?
- [ ] Distill the month's biggest lesson into `MEMORY.md` → "Winning Moves" or "Key Decisions"

## After Deal Close or Loss (Anytime)

- [ ] Update `leads/PIPELINE.md` with final status
- [ ] Write post-mortem in today's daily log (what worked or what went wrong)
- [ ] Update `LESSONS.md` with relevant pattern or mistake
- [ ] If client intel was gathered → update `LESSONS.md` → "Client Playbook"
- [ ] If deal was lost → log the reason. Price? Timing? Ghosted? Competitor? This data is gold.

---

_Checking boxes isn't the point. Getting better is the point. Every check is a chance to learn something._
